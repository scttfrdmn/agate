# Screenshots

Drop product screenshots here and reference them from the top-level `README.md`
("A quick tour" section). Suggested set (from the live demo):

- `ask.png` — an Ask answer with citations + the cost receipt.
- `notebook.png` — a notebook with a code cell rendering a matplotlib plot inline.
- `context.png` — the Context panel (send window / clear / compress) + running cost meter.

Keep them reasonably sized (≤ ~200 KB each; PNG). They are the first thing a repo
browser sees of the actual product, so favor a clean, populated screen over an empty one.
